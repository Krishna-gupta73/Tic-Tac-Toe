# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. This project features a clean and responsive interface where two players take turns playing as **O** and **X**. The game automatically detects the winner or a draw and allows players to restart and play again.

## 🚀 Live Demo

🔗 **Play Here:** https://krishna-gupta73.github.io/Tic-Tac-Toe/

## ✨ Features

- 🎯 Player **O** starts the game
- 🎮 Two-player gameplay
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Reset game functionality
- 📱 Responsive design
- ⚡ Built with vanilla JavaScript (No frameworks)

## 🎮 How to Play

1. Player **O** starts the game.
2. Players take turns selecting an empty cell.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click **Reset Game** to start a new match.
